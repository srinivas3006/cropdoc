Step 1: Set up your folders (Crucial First Step)
Before doing anything, you need to add your images. 
Inside `dataset_splits/train/`, create a folder for EACH disease or vegetable you want the AI to learn.
Example: `dataset_splits/train/Potato_Healthy/` and `dataset_splits/train/Potato_Blight/`.
Paste your .jpg images into their matching folders. 
The AI will automatically count these folders and learn their names!

Step 2: Install all required Python packages (Only need to do this once)
pip install -r requirements.txt

Step 3: Train the neural network model on the dataset
python src/train.py --epochs 50 --color_mode rgb
(What this does: It automatically counts your folders, builds a custom brain, feeds all the images into it, and saves a `class_names.json` dictionary alongside your trained model.)

Step 4: View the training progress graphs (Open http://localhost:6006 in your browser)
tensorboard --logdir logs/

Step 5: Test the trained model on a single unseen image to get a prediction
python src/predict.py --image_path "path/to/test.jpg" --model_path "saved_models/best_model_rgb.keras"
(What this does: It secretly reads the `class_names.json` dictionary generated during Step 3, looks at your test image, and prints the exact folder name it thinks the image belongs to!)
